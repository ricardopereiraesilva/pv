# Ícones PWA

Esta pasta deve conter os ícones do aplicativo nos seguintes tamanhos:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Como Gerar os Ícones

1. Crie uma imagem quadrada de 512x512 pixels com o logo do aplicativo
2. Use uma ferramenta como [RealFaviconGenerator](https://realfavicongenerator.net/) para gerar todos os tamanhos
3. Coloque os arquivos gerados nesta pasta

## Dica Rápida

Você pode criar um ícone simples usando uma imagem do YouTube ou criar um design personalizado que represente seu aplicativo de playlist.
